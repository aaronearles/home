# Coder with Claude Code Workspaces

Self-hosted development environment with secure Claude Code workspaces behind Traefik.

## Features

- **Coder Control Plane**: Web-based workspace management
- **Claude Code Integration**: Pre-configured workspaces with Claude Code CLI
- **Traefik Integration**: SSL termination and reverse proxy
- **Egress Firewall**: Network isolation allowing only Coder + Anthropic API
- **Managed Security**: Locked-down Claude Code settings preventing sensitive file access
- **Persistent Workspaces**: Home directories stored in Docker volumes

## Prerequisites

- Docker Engine with Compose plugin
- Traefik deployed and running (`../traefik`)
- `traefik` network must exist

## Quick Start

```bash
# 1. Create environment file
cp .env.sample .env
# Edit .env and set POSTGRES_PASSWORD

# 2. Deploy Coder
./run.sh

# 3. Access at https://coder.internal.earles.io
# Complete admin setup in browser

# 4. Create workspace from "Claude Code (Docker)" template
```

## Architecture

```
docker-compose.yml
├── coder-db (postgres:17)
│   └── Backend network only
└── coder (ghcr.io/coder/coder:latest)
    ├── Traefik network (external access)
    └── Backend network (database access)
```

### Networks

- **traefik**: External network for reverse proxy (shared)
- **backend**: Internal network for database communication

### Volumes

- **coder_db_data**: PostgreSQL database storage

## Configuration

### Environment Variables

See `.env.sample` for all configuration options.

**Required:**
- `POSTGRES_PASSWORD`: Database password (auto-generated by run.sh)
- `DOCKER_GID`: Docker socket GID (auto-detected by run.sh)

**Optional:**
- `CODER_OIDC_*`: EntraID/OIDC SSO configuration

### Traefik Labels

- **Domain**: `coder.internal.earles.io`
- **Entrypoint**: `websecure` (HTTPS)
- **Certificate**: Let's Encrypt via Cloudflare DNS
- **Port**: 7080 (internal)

## Workspace Template

The included Terraform template (`template/main.tf`) creates workspaces with:

### Security Features

1. **Managed Settings** (`/etc/claude-code/managed-settings.json`)
   - Blocks sensitive file access (.env, .key, .pem, etc.)
   - Disables external network tools (curl, wget, ssh)
   - Requires approval for destructive operations

2. **Egress Firewall** (iptables)
   - Allows: Coder control plane, Anthropic API, DNS
   - Blocks: All other outbound traffic
   - TODO: Remove npm/https once Claude Code is pre-baked

3. **Network Isolation**
   - Workspaces run in bridge network
   - No direct host network access
   - Controlled egress via iptables

### Included Tools

- **Claude Code**: Installed via npm (TODO: pre-bake in image)
- **code-server**: VS Code in browser on port 13337
- **Standard dev tools**: git, curl, etc.

## Deployment

### Initial Setup

```bash
./run.sh
```

The script will:
1. Check prerequisites (Docker, Traefik network)
2. Generate .env with random password
3. Deploy services and wait for health checks
4. Install Coder CLI
5. Login and push workspace template

### Update Deployment

```bash
docker compose pull
docker compose up -d
```

### View Logs

```bash
docker compose logs -f coder
docker compose logs -f coder-db
```

## Troubleshooting

### Coder Not Accessible

Check Traefik labels and network:
```bash
docker inspect coder | grep -A5 Labels
docker network inspect traefik
```

### Database Connection Issues

Check health status:
```bash
docker inspect coder-db --format "{{.State.Health.Status}}"
docker logs coder-db
```

### Workspace Connection Issues

Check workspace logs in Coder UI or:
```bash
docker logs coder-<username>-<workspace>
```

## Security Hardening

### Production Checklist

- [ ] Build custom image with Claude Code pre-installed
- [ ] Remove npm/https egress rules (lines 304-305 in main.tf)
- [ ] Uncomment `iptables -P OUTPUT DROP` (line 308 in main.tf)
- [ ] Enable OIDC SSO in .env
- [ ] Pin Coder image to specific version (not `latest`)
- [ ] Add monitoring to Gatus
- [ ] Review and audit managed-settings.json rules

### OIDC Integration

Add to `.env`:
```bash
CODER_OIDC_ISSUER_URL=https://login.microsoftonline.com/<tenant>/v2.0
CODER_OIDC_CLIENT_ID=<your-client-id>
CODER_OIDC_CLIENT_SECRET=<your-client-secret>
CODER_OIDC_EMAIL_DOMAIN=yourdomain.com
```
